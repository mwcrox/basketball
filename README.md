# 🏀 Conference Tournament Tracker

A responsive, basketball-themed web app that tracks all 31 NCAA Division I Men’s Basketball conferences that receive an automatic bid to the 2026 NCAA Tournament.

The site dynamically organizes conferences based on the tournaments that are going on today. Users can also pick a day in March to see upcoming or past games. 

---

## 🚀 Features

### 🏆 Smart Conference Categorization

Conferences automatically appear in one of four sections:

1. **Conference Championship Today**
2. **Conference Tournament Has Started**
3. **Conference Tournament Schedule (Not Started Yet)**
4. **Conference Tournament Over**

Empty sections are automatically hidden.

---

### 📅 March-Only Date Simulation

- Select any day in **March 2026**
- Automatically re-sorts all 31 conferences
- “Today” option uses the viewer’s real local date
